# HearAdjust

中文 | [English](README.md)

HearAdjust 是一个 Chrome 扩展，用于实时模拟听力损失，并提供类似助听器的音频调节能力。它既可以让健听者短暂体验不同听力状况下的声音世界，也可以让用户通过简单的 8 频段均衡器做个性化补偿。

## 功能

### 共情体验模式

使用 Web Audio 实时模拟不同类型的听力状况。

| 预设 | 说明 |
| --- | --- |
| 轻度听力损失 | 高频轻度衰减，在嘈杂环境中更难听清细节 |
| 中度听力损失 | 从语音关键频段开始明显衰减，辅音理解更困难 |
| 重度听力损失 | 全频段大幅衰减，只剩非常响的声音较容易感知 |
| 年龄相关性听力损失 | 高频逐步下降，背景噪声中语音更难分辨 |
| 噪声性听力损失 | 长期暴露噪声后常见的 4kHz 凹陷 |
| 耳鸣 | 在正常听觉上叠加持续 6kHz 音调 |
| 低频听力损失 | 低频感知变弱，而高频相对更清晰 |
| 自定义听力图 | 输入你自己的损失曲线，让别人听到你平时听到的声音 |

### 助听调节模式

手动调节 8 频段 EQ，或者从快速预设开始：

- 平坦（重置）
- 语音清晰
- 高频增强
- 低频增强

设置会保存在本地，下次打开弹窗时自动恢复。

## 安装方式

当前项目以开发版扩展方式加载。

1. 克隆或下载本仓库。
2. 在 Chrome 中打开 `chrome://extensions`。
3. 打开开发者模式。
4. 点击“加载已解压的扩展程序”。
5. 选择项目根目录。

由于使用了 Manifest V3 和 offscreen API，需要 Chrome 116 或更高版本。

## 使用方法

### 基本流程

1. 打开一个正在播放音频或视频的标签页。
2. 点击 HearAdjust 扩展图标。
3. 当前版本一次只能处理一个标签页。
4. 将三态开关拨到左侧进入共情模式，或拨到右侧进入助听模式。
5. 选择预设，或自行调节滑块。
6. 如果要切换到别的网页，请先把开关拨回中间关闭当前处理。

### 关闭状态

当开关停在中间时，弹窗会显示两个模式的说明，而不是继续停留在上一次的控制面板。

### 共情体验模式

- 点击任意听力状况卡片即可立即生效。
- 选择“自定义听力图”后，可以在 8 个频段上填写自己的损失曲线。
- “耳鸣”预设会额外叠加持续正弦音，模拟主观耳鸣。

### 助听调节模式

- 快速预设可一键加载常见补偿曲线。
- 8 个滑块范围为 `-40dB` 到 `+40dB`。
- 手动拖动滑块后，当前快速预设高亮会被取消。

## 注意事项

- 当前版本同一时间只能处理一个标签页；如果要切换到其他网页，请先关闭当前标签页的处理。
- 某些带 DRM 的网站无法通过 `tabCapture` 捕获音频。
- 重度类预设会显著降低音量，建议谨慎测试。

## 技术结构

```text
popup.html / popup.js
  -> background.js（service worker）
  -> offscreen.js（offscreen document）
  -> audioProcessor.js（EQ 滤波器 + 耳鸣振荡器）
```

## 技术栈

- Manifest V3 Chrome 扩展
- Web Audio API
- `chrome.tabCapture`
- `chrome.offscreen`
- 原生 JavaScript + ES Modules

## 项目背景

HearAdjust 不是医疗设备。它更像一个关于共情和可及性的交互式小项目，希望帮助更多人理解听力损失，并探索声音适配的可能性。
